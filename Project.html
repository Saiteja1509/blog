Project 2

Vechile Detection and Counting

Vehicle detection and counting are essential tasks in transportation engineering and intelligent transportation systems. In this project, we present a system for detecting and counting vehicles in real- time video streams using OpenCV in Python. Our system uses a deep learning-based object detection model, YOLOv3, to detect vehicles in each frame of the video. We then track the detected vehicles using the DeepSORT algorithm, which is a simple online and real-time tracking algorithm. Finally, we count the number of vehicles in each frame and calculate the traffic flow rate.
